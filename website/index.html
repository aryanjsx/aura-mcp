What AURA Does

AI Interpretation

Describe a task in plain English. AURA converts it into a structured execution plan using OpenAI or a local model.

Project Scaffolding

Generates fully functional React, Node.js, and FastAPI projects with real, runnable code.

Notion Integration

Reads tasks from your Notion database, executes them, and writes results back — fully automatic.

GitHub & Docker

Create repos, push files, and generate Dockerfiles — all through the plugin system.

Plugin Architecture

Drop a Python file into the plugins folder and it's live. No core changes needed.

MCP Compatible

Works with Claude Desktop and any MCP client via the standard stdio transport.

Quick Install

pip install -e .
aura doctor
