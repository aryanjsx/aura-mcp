Everything you need to ship faster

AURA handles the boring parts so you can focus on what matters.

🧠

AI Interpretation

Describe a task in plain English. AURA parses it with OpenAI and falls back to a deterministic rule engine when no API key is set.

🛫

Safety Validation

Every plan is validated and sanitized before execution. Malformed or dangerous input never reaches your filesystem.

📦

Project Scaffolding

Generates complete, runnable projects with proper folder structure, dependencies, routes, and configuration files.

📋

Notion Integration

Reads pending tasks from your Notion database, builds the project, and marks them done automatically.

🔌

Plugin System

Five built-in plugins with auto-discovery. Drop a Python file into the plugins folder and it's live on restart.

🤖

MCP Compatible

Works natively with Claude Desktop, Cursor, and any MCP-compatible AI client over stdio transport.

How it works

From idea to project in three steps.

1

Describe

Write a task in Notion, type a prompt in Claude Desktop, or send a command via the CLI.

"Build a FastAPI todo app with authentication"
→
2

Interpret

AURA parses your intent, selects the right framework, and creates a validated execution plan.

action: scaffold_project, framework: fastapi, name: todo_app
→
3

Build

Real files appear on disk — routes, models, config, dependencies. Ready to run.

Created 6 files in ~/projects/todo_app/
